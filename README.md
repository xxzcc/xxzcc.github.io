# Chenchen Zhang Personal Website

This is a static GitHub Pages starter for `https://xxzcc.github.io`.

## Deploy

1. Create a public GitHub repository named `xxzcc.github.io`.
2. Copy these files into that repository.
3. Push to the `main` branch.
4. In GitHub, open Settings -> Pages and publish from the `main` branch.

## Edit

- Main content: `index.html`
- Styling: `assets/css/main.css`
- Dark mode script: `assets/js/main.js`

The page currently uses `zcc1959339538@gmail.com` as the public email address.

Visitor geography requires a site-specific map widget token, such as ClustrMaps,
PulseMaps, or RevolverMaps. Replace the footer visitor-map placeholder in
`index.html` after registering the public domain.
